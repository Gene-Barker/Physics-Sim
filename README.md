# Dependencies
#### Any JDK, such as openjdk

# Installation
```text
1. git clone https://github.com/Gene-Barker/Physics-Sim
2. cd Physics\ Engine
3. chmod +x PhysicsSim.sh
4. ./PhysicsSim.sh
```
